# Sistem de Gestiune Contabilitate

Aplicație web completă pentru gestionarea clienților și a fluxului de lucru contabil.

## Caracteristici
- **Backend (Contabil)**: Dashboard, Gestiune Clienți (CRUD), Gestiune Utilizatori (Admin).
- **Frontend (Client)**: Portal dedicat pentru vizualizarea documentelor și comunicare.
- **Roluri**: Admin, Contabil Șef, Contabil, Jurist.
- **Securitate**: RLS (Row Level Security) la nivel de bază de date.

## Tehnologii
- React + Vite
- Tailwind CSS
- Supabase (PostgreSQL + Auth)
- Lucide Icons
